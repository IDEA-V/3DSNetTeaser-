# 3DSNetTeaser++
### Install
If you don't have PCL, install it by
```
./install_pcl.sh
```
For C dependencies
```
./install_c.sh
```
Install 3DSmoothNet and TEASER-plusplus and compile them.
```
./install.sh
```

### Usage
Run pipeline.py with number of threads you want.
```
 OMP_NUM_THREADS=16 python pipeline.py
```
